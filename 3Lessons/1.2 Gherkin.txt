Функция: Нужно создать консольное меню, которое будет выполнять действия по нажатию цифр.
    Сценарий: Вывод списка 
        Дано консольное меню которое выполняет действия по нажатию цифр
        Когда нажимают на цифру 1 (вывод списка)
        То выводиться список всех форматов
    Примеры:
        |    Короткое описание   |          Полное описание         |                    
        |          png           |    portable network graphics     |
        |          txt           |               text               |
        |          jpg           | joint photographic experts group |
        |          gif           |    graphics interchange format   |
        |          doc           |            document              |

Функция: Нужно создать консольное меню, которое будет выполнять действия по нажатию цифр.
    Сценарий: Добавление нового формата 
        Дано консольное меню которое выполняет действия по нажатию цифр
        Когда нажимают на цифру 2 (добавление нового формата)
        То добавляется новый формат
    Примеры:
        |Введите короткое описание формата:  | Введите полное описание: |
        |            exe                     |       executable         |
         Формат будет добавлен
