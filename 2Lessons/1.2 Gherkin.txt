Функция: Вывести симметрична ли квадратная матрица относительно главной диагонали?
    Сценарий: Вывод симметрична ли квадратная матрица
        Дано ввод размера массива и заполнение его
        Когда вводят <arraySize> и заполняют <array>
        То выводиться <result>
    Примеры:
        | arraySize |  array  |        result          |
        |     2     |  1   0  |   Матрица симметрична  |
        |           |  0   1  |                        |
        |     2     |  3   6  | Матрица не симметрична |
        |           |  3   6  |                        |
